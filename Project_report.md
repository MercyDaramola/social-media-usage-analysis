# Social Media Usage and Student Academic Performance Analysis Using Interactive Data Visualization Techniques
Abstract

This project investigates how social media usage influences students' academic performance. Using survey data collected from university students, Power BI was used to clean, transform, analyze, and visualize the data. Interactive dashboards were developed to identify patterns in social media usage, study habits, academic performance, and student behavior.

# Introduction

Social media has become an essential part of students' daily lives. While it provides opportunities for communication and access to educational resources, excessive usage may reduce study time, interrupt concentration, and affect academic performance.

This project explores these relationships using interactive data visualization techniques to support better decision-making.

# Objectives

The objectives of this project are:

- Analyze students' social media usage patterns.
- Determine the most popular social media platforms.
- Compare study hours with social media usage.
- Analyze the relationship between social media usage and CGPA.
- Evaluate the educational benefits of social media.
- Identify the effects of notifications and sleep disruption.
- Develop an interactive Power BI dashboard for data exploration.

Key Findings

# The analysis revealed several important insights:

WhatsApp was the most frequently used social media platform among students.
Most students reported using social media during their free time and before sleeping.
A significant proportion of respondents experienced notification-related distractions while studying.
Many students acknowledged that social media affected their sleep quality.
Social media also contributed positively by supporting collaborative learning, improving communication, and providing access to educational resources.
The correlation between social media usage and CGPA was -0.01, indicating a very weak negative relationship between time spent on social media and academic performance within this dataset.
Conclusion

The findings indicate that social media has both positive and negative effects on students. Although it can distract student and affect sleep, it also serves as an important tool for collaboration, communication, and accessing educational materials. The weak correlation between social media usage and CGPA suggests that academic performance is influenced by multiple factors beyond social media alone.
